# 学生成绩管理系统 - 完整项目文件
## 项目概览
这是一个完整的学生成绩管理系统项目，基于HTML5 + CSS3 + JavaScript开发，具备以下功能：
### 🎯 核心功能
- 📊 **学生管理** - 添加、编辑、删除学生信息
- 📈 **成绩管理** - 录入、查询、修改学生成绩
- 📋 **统计分析** - 成绩分布图表、科目平均分分析
- 💾 **数据持久化** - 基于LocalStorage的本地存储
- 📱 **响应式设计** - 适配各种设备屏幕
### 🛠 技术栈
- **前端框架**: 原生JavaScript (ES6+)
- **UI框架**: Bootstrap 5
- **图表库**: Chart.js
- **图标库**: Font Awesome
- **存储方式**: LocalStorage

## 🚀 快速开始
### 1. 本地运行
```bash
# 方式1: 使用Python
cd student_grade_071
python -m http.server 8080
# 方式2: 使用Node.js
npx http-server -p 8080
# 方式3: 直接在浏览器中打开（可能有CORS限制）
# 直接双击 index.html
```
访问地址: `http://localhost:8080`
### 2. GitHub部署
1. 将项目上传到GitHub仓库
2. 启用GitHub Pages
3. 访问生成的URL
